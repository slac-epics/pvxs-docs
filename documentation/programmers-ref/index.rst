Programmers Reference
=====================

For application developers using pvxs (C++), P4P (Python), or phoebus
(Java) PVA APIs to write Secure-PVAccess-aware software. Covers library
configuration, environment variables consumed by the library, certificate
management from the application side, and EXPERT API additions.
