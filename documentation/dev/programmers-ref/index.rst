Programmers Reference
=====================

For application developers using pvxs (C++), P4P (Python), or phoebus
(Java) PVA APIs to write Secure-PVAccess-aware software. Start here if
you need to build the libraries, link your own executables, write an
IOC, server, or client application, build an authenticator, or understand
the monitoring and performance hooks that SPVA adds around the protocol.
